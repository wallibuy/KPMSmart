QUICK START GUIDE
================

1. EXTRACT FILES: Unzip this folder to your desired location

2. INSTALL PYTHON: Make sure you have Python 3.7+ installed
   Check with: python --version

3. RUN THE APPLICATION:
   Option A (Recommended): Double-click "run.py" 
   Option B: Open terminal/command prompt and type: python run.py
   Option C: Type: python app.py

4. OPEN IN BROWSER: Go to http://localhost:5000

3. Check if port 5000 is available
